# T.O.D Music Player

## Overview

T.O.D Music Player is a web-based music player developed for the CodeClause internship. It demonstrates essential music player functionalities and showcases web development skills.

## Features

- Play, pause, and control music.
- Navigate between tracks.
- Random track selection.
- Repeat track functionality.
- Volume control.
- Dynamic background based on the playing track.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/CodeClause_Music-Streaming-Application.git

## Usage

1. **Open the project directory:**
    ```bash
    cd CodeClauseInternship_Music-Streaming-Application
    ```

2. **Open `index.html` in your web browser.**

3. **Explore the T.O.D Music Player:**
    - Open the player in your web browser.
    - Play, pause, and navigate between tracks.
    - Adjust the volume using the slider.
    - Experience dynamic background changes.
